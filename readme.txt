== Description ==

A simple plugin to let editor add Post Ideas to the database and for Contributor to assign the Post Ideas to themselves

== Installation ==

1. Upload this plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Post Ideas -> Add Post Ideas to start adding Post Ideas.
